Text Settings

The text settings for CSS can be broken down into two categories, those setings that relate to the text itself, and those settings that relate to the physical appearance of the text through font settings

Text alignments

The text location on the page can be manipulated in several ways simply through settings directly addressed to the text.

  • text-color allows a developer to change the color of the text. The rule of thumb is to make sure it contrasts with the color behind the text.
  • The text alignment traits provide several methods to direct the location of the text, including text-align; text-align-last; direction; unicode-bidi; and vertical-align.
  • The text decoration traits provide for lines around the text, whether underlining the text, striking through the text, or placing a line above. The traits include text-decoration-line; text-decoration-color; text-decoration-style; and text-decoration-thickness.
  • The text spacing attributes adjusts the amount of space between text and other objects, including the text itself. Attributes include letter-spacing, line-height, text-indent, white-spacing, and word-spacing.

    Font settings

    The font settings handle the appearence of the text.

    The basic settings are covered under the font-family attribute. All font families fall into thefive basic catagories:

  • serif,
  • sans-serif,
  • monospace,
  • cursive, and
  • fantasy.
